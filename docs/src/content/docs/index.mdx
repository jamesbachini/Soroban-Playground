SoroPG is a browser-based development environment for Stellar smart contract developers. It combines a Rust editor, multi-file workspaces, Docker-backed build and test execution, wallet tooling, contract deployment, and an interface explorer for deployed contracts.

## Typical Workflow

1. Create or import a workspace.
2. Edit `Cargo.toml`, `src/lib.rs`, and tests in the browser.
3. Run unit tests and Scout audit from the **Test** panel.
4. Compile the contract from the **Build** panel and download the `.wasm`.
5. Deploy the downloaded `.wasm` from the **Deploy** panel.
6. Load the new contract ID in **Explore** to inspect and call methods.

## When to Use SoroPG

Use SoroPG when you want a fast way to prototype Stellar smart contracts without setting up a complete local Stellar development environment. The editor stores projects in your browser, while build and test commands run on the SoroPG server inside an isolated Docker sandbox.

For production deployments, keep your source code in version control, review generated transactions carefully, and test against the same network you plan to use.

## What Runs Where

- Browser: editor state, workspaces, generated local wallet keys, Stellar SDK transaction building, and wallet signing requests.
- SoroPG server: static frontend/docs hosting, Docker sandbox orchestration, build/test/audit streams, contract interface CLI calls, and local friendbot proxying.
- Stellar network: RPC/Horizon reads, transaction submission, account funding, and deployed contract execution.

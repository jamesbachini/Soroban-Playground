SoroPG is a browser-based Soroban playground for Stellar smart contract developers. It gives you a Rust editor, project workspaces, Docker-backed build and test execution, wallet tooling, contract deployment, and an interface explorer for deployed contracts.

## What SoroPG is for

Use SoroPG when you want a fast way to prototype Soroban contracts without setting up a complete local Stellar development environment. The editor stores projects in your browser, while build and test commands run on the SoroPG server inside an isolated Docker sandbox.

For production deployments, keep your source code in version control, review generated transactions carefully, and test against the same network you plan to use.

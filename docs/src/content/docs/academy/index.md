SoroPG Academy is a guided learning track inside the SoroPG editor. The editor stays open while the Academy panel gives you the next course step, links to written tutorials, and routes you into the existing Create, Test, Build, Deploy, Explore, AI, and MCP tools.

Academy progress is stored in your browser. Completing a course on one browser or device does not automatically sync to another.

## Available courses

- [Hello World: Build, Test, and Deploy](./hello-world-build-test-deploy/) teaches the full first contract loop: import code, read the Rust contract, run tests, build WASM, deploy to Testnet, verify the contract, and invoke it.
- [AI-Assisted Contract Development](./ai-assisted-contract-development/) shows how to use the built-in assistant as a focused coding partner while keeping every change reviewable.
- [SoroPG + Agentic AI via MCP](./agentic-ai-mcp/) explains how to connect Codex, Claude Code, or another MCP client to the live browser workspace.

## How to use Academy

1. Open the **Academy** tab in SoroPG.
2. Choose a course from the chapter list.
3. Keep the editor visible while you follow the course home page in the bottom panel.
4. Open the written tutorial in a new tab when you need deeper context.
5. Use the course action buttons to load material, run tests, build, deploy, or open AI tooling.
6. Return to Academy to verify or mark the course complete.
